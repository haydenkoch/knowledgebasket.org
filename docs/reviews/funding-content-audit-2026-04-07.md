## Implementation Notes

- Source of truth for the replacement catalog now lives in `scripts/data/california-tribal-funding.mjs`.
- Live database sync is handled by `scripts/sync-curated-funding.mjs`.
- The shared funding sync helper now rebuilds the `funding` Meilisearch index from the published funding table after each curated sync, so browse and search stay aligned with the database.
- `scripts/seed-coils.js` now uses the same curated dataset and pruning logic, so `pnpm db:seed` no longer restores the synthetic federal-heavy list.
